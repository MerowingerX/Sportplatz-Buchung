# Einmalige Buchung in Serienbuchung wandeln

Für den Admin soll ein Feature zur Verfügung stehen, eine beliebige Buchung in eine Serienbuchung zu wandeln. Also die Daten zu übernehmen, in den Serienbuchungstermineditor anzubieten. Er bestätigt dann noch (nach möglichen Änderungen) und der normale Termin wird zur Serie.

Die originale Buchung bleibt dann einfach bestehen. Die Serie wird dann für diesen Tag nicht buchen können, aber das macht nix. Die buchung gehörtz ja auch ggf jemand anderem
